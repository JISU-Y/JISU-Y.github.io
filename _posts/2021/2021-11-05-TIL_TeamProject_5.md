## 오늘 배운 것

### git

git push branch 이렇게 해야함

바로 push master 하면 내가 다른 사람들거를 다 수정하지 않는 이상 내걸로만 덮어씌어지게 된다

그러니까 항상 push branch로 하고 pull requset하고 merge할것

git rebase 도 있다.
